## Installation

1. Clone or download the project files
2. Ensure all files are in the same directory:
   - `auto-id-card-generator-tool.html`
   - `auto-id-card-generator-tool.css`
   - `auto-id-card-generator-tool.js`
3. Open `auto-id-card-generator-tool.html` in a web browser

## Usage

### Basic ID Card Creation
1. Select a template from the Design section
2. Fill in personal information
3. Upload photos and logos
4. Customize colors and fonts
5. Preview and download

### Advanced Features
- **Batch Generation**: Upload CSV files for multiple cards
- **AI Enhancement**: Use AI to improve design and content
- **Security Features**: Add barcodes, QR codes, and holograms
- **Back Side Content**: Add rules and terms for card back

### Export Options
- **Single Card**: Download individual cards as PDF, PNG, or JPG
- **Batch Export**: Download multiple cards as ZIP file
- **Print**: Direct printing with proper formatting
- **Canva**: Export to Canva for advanced editing
